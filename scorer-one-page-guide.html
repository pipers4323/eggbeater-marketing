One-Page Scorer Guide

Live Scoring, without the extra noise

Use this guide for coaches, team managers, and designated scorers. The goal is simple: enter the right game, keep the score accurate, record the key events, and close the game cleanly.

What Eggbeater is for

  • Spectators do not need to sign in. They can open the app and follow schedule, scores, history, roster, and tournament updates.
  • Only the scorer uses scorer mode. The scorer password unlocks live scoring controls for the assigned game.
  • Keep the score right first. If anything gets hectic, accurate score and game state matter more than perfect stat detail.

Before the game

Open the app, go to Scores, and unlock scorer mode.

  • Tap Login to Score
  • Enter the scorer password
  • Tap Open Scorer on the correct game
  • Confirm opponent, time, and cap color
  • Confirm roster/player names look right
2

During the game

Use the live controls to track the game as it happens.

  • Tap Start when play begins
  • Record goals, assists, steals, exclusions, and saves
  • Track opponent goals too
  • Update quarter / halftime / game-state changes
  • If unsure, keep the score accurate first
3

After the game

Close the game cleanly so spectators see the correct final result.

  • Set the game state to Final
  • Confirm the final score
  • Select the correct result
  • Share/send the box score if needed
  • Lock scorer mode when finished

Best Practices

  • Score one game only. Make sure you are in the correct game before entering events.
  • Pause if something looks wrong. Refresh and verify before continuing if the score, game, or roster looks off.
  • Use the game-state buttons consistently. Quarter and final state changes are part of what spectators rely on.
  • Do not share scorer access broadly. Keep the scorer password with designated scorers only.

Fast Troubleshooting

If it is working

You should see the correct game card, cap color, live score controls, and an updated score as events are entered.

If something looks off

  • Refresh the app
  • Re-open the correct game
  • Check scorer mode is still unlocked

If something breaks

Send: what you were trying to do, what happened instead, your device type, and a screenshot if possible.
